Aide myIATA

Page d'accueil du Portail myIATA

Voici la page d'accueil du portail myIATA. Le portail comporte plusieurs icônes, outils de navigation et menus. Ils vous permettent d'accéder aux différentes fonctionnalités à votre disposition .

L'entête contient les outils suivants :

Votre nom  : cliquez sur ce lien pour visualiser et modifier vos coordonnées.
Nom de votre société  : cliquez sur ce lien pour visualiser les informations relatives à votre entreprise.
Imprimer : cliquez sur ce lien pour imprimer le contenu de la page. Une boîte de dialogue d'impression s'ouvre, cliquez sur le bouton Imprimer pour imprimer la page.

Recherche : cliquez sur ce lien pour rechercher des informations relatives à IATA.

  • Cliquez sur Recherche , saisissez ce que vous recherchez dans le champ texte qui s'affiche à l'écran, cliquez sur Aller pour lancer la recherche.
  • La page Web des résultats de recherche Google s'ouvre automatiquement et vous donne tous les résultats de la recherche.
  • Lorsque vous cliquez sur n'importe lequel de ces liens, vous revenez au site web IATA et les informations requises s'affichent.

Aide : si vous avez besoin d'aide sur l'un des écrans du portail, cliquez sur ce lien.
Quitter : Cliquez sur ce lien pour vous déconnecter du portail et revenir à l'écran de connexion.

Applications liées au portail : Approvisionnement en billets, Factures, Services Clients sont les différentes applications externes que vous pouvez voir et utiliser en fonction des autorisations définies par votre administrateur.

La zone Info, à gauche de la page, affiche les informations suivantes :

Code IATA : le code IATA de la société actuellement sélectionnée.
Nom de la Société : le nom de la société actuellement sélectionnée.
[Modifier]  : cette option vous permet de sélectionner la société affiliée à votre compagnie pour laquelle vous avez l'intention d'effectuer des transactions ou de travailler.

Administrateurs de Compte : si vous avez besoin d'une aide ou d'une assistance quelconque ou si vous avez des questions relatives à l'utilisation du portail, vous pouvez contacter vos Administrateurs de Compte. .

Vous pouvez utiliser les menus situés sous la Boîte d'Infos pour accéder aux différentes zones du Portail.

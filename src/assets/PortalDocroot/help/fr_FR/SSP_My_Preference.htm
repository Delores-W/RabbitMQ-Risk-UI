Mes Préférences

Vous pouvez définir les préférences de langue dans cet écran.

Cliquez sur la liste déroulante Choisir une Langue et sélectionnez une langue de votre choix.

Cliquez sur Enregistrer les Modifications pour définir votre langue préférée pour le portail. Cliquez sur Annuler les Modifications pour revenir aux préférences de langue enregistrées précédemment.

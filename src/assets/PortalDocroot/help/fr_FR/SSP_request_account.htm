Aide myIATA

Demander un Compte

Cet écran vous permet de demander un compte dans le portail myIATA.

Nom de la Société : saisissez le nom de votre société dans ce champ.

Code Numérique IATA : saisissez le Code IATA dans ce champ.

Ligne Adresse 1 : saisissez l'adresse du contact qui sera utilisée par IATA pour la correspondance.

Pays : sélectionnez votre Pays dans la liste déroulante.

Etat / Province : sélectionnez votre état/province dans la liste déroulante.

Nom du Contact : saisissez le nom du contact dans ce champ.

E-mail : saisissez une adresse e-mail valide. Votre nom d'utilisateur et votre mot de passe provisoires vous seront envoyés à cette adresse e-mail.

Type de Client : sélectionnez le Type de Client dans la liste déroulante.

Cliquez sur Soumettre . Votre nom d'utilisateur et votre mot de passe vous seront envoyés par e-mail après le traitement de votre dossier.

Si vous souhaitez effacer tous les champs pour recommencer la saisie du formulaire, cliquez sur Réinitialiser .

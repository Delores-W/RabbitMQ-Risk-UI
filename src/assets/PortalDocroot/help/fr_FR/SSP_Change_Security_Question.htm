Modifier la Question de Sécurité

Question secrète : choisissez l'une des questions dans la liste déroulante des questions secrètes. Si vous oubliez le mot de passe ultérieurement, vous devrez choisir la même question afin de récupérer votre mot de passe. Choisissez une question facile et mémorisez la question que vous avez choisie dans ce champ.

Réponse à la question secrète : saisissez la réponse à la question choisie dans ce champ. La réponse doit contenir un minimum de quatre caractères. Si vous oubliez votre mot de passe ultérieurement, vous devrez répondre à la question afin de récupérer le mot de passe. Mémorisez donc la réponse que vous avez saisie dans ce champ.

Mot de passe : saisissez votre mot de passe dans ce champ.

Cliquez sur Enregistrer les Modifications pour mettre à jour la question secrète ou sur Annuler les Modifications pour revenir aux réglages d'origine.

Mot de passe oublié

Cet écran vous aide à récupérer votre mot de passe.

Nom d'utilisateur : saisissez votre nom d'utilisateur dans ce champ.

Question de Sécurité : sélectionnez la question de sécurité que vous avez définie dans la page Modifier la Question de Sécurité dans le Portail.

Réponse de Sécurité : saisissez la réponse que vous avez donnée dans la page Modifier la Question de Sécurité dans le Portail.

Cliquez sur Soumettre . Votre mot de passe vous sera envoyé par e-mail.

Si vous souhaitez effacer tous les champs, cliquez sur Réinitialiser .

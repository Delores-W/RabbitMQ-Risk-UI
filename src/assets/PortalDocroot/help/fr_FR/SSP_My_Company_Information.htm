Aide myIATA

Informations relatives à ma Société

Cet écran vous permet de visualiser et de mettre à jour les informations relatives à la société pour laquelle vous travaillez.

Les champs Nom de la Société, Société Mère, Adresse de la Société, Code IATA et le Type de Société ne peuvent pas être modifiés.

Téléphone : lorsque vous saisissez le numéro de téléphone dans ce champ, assurez-vous d'entrer le code de pays dans la première zone, suivi du préfixe de région et du numéro de téléphone dans les deux zones suivantes.

Fax : lorsque vous saisissez le numéro dans le champ Fax, respectez le format que vous avez utilisé pour la saisie du numéro de téléphone.

E-mail : ce champ doit contenir l'adresse e-mail de la société.

N° de TVA du client : saisissez votre numéro de TVA.

Mode de Livraison de la Facture : ce champ indique votre choix concernant la réception de la confirmation de facture. Si vous souhaitez recevoir la confirmation de facture par e-mail, cliquez et cochez la case attenante à Facture électronique.

Cliquez sur Enregistrer les Modifications pour mettre à jour les informations relatives à la Société. Cliquez sur Annuler les Réglages pour revenir aux informations enregistrées précédemment.

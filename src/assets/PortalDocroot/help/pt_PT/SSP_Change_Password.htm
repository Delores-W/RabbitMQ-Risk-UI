Ajuda - myIATA

Alterar password

Poderá alterar a sua password neste ecrã.

Password antiga: Introduza a sua password actual neste campo.

Nova password: Introduza a sua nova password no campo Nova password. A palavra-passe pode ter entre 8 e 10 caracteres. Os primeiros sete caracteres deverão incluir pelo menos 1 letra maiúscula. Deverá haver pelo menos 1 número depois dos 7 primeiros caracteres. Por exemplo Myiata06

Introduza novamente a sua nova password: Confirme a nova password, introduzindo novamente a nova password neste campo.

Clique em Alterar password para definir a nova password.

Clique em Cancelar alterações para limpar todos os campos.

Todos os campos são de preenchimento obrigatório; assim, apenas poderá alterar a password se souber a password actual. Recomenda-se a alteração da password pelo menos uma vez de três em três meses.

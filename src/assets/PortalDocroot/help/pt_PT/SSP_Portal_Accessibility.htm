Ajuda - myIATA

Acessibilidade ao portal

Este ecrã apresenta uma lista de todas as aplicações T2005.

O acesso a qualquer uma das aplicações apresentadas é definida pelo administrador ao configurar a conta.

A coluna aplicação enumera as aplicações T2005.
A coluna função enumera as funções que lhe foram atribuídas para cada uma destas aplicações.

O acesso à aplicação é indicado na coluna On/Off . Indica que o acesso é autorizado, enquanto indica que o acesso é proibido.

Profile Management

This Screen allows you to add, edit and delete profiles. These profiles are later assigned to users at the time of creation / update.

Add Profile

Select Add New Role from the Select User Role drop down.

Assign New role name in the New Role Name Field .

Select the Role type from the Role Type dropdown.

Add the Decription for the new profile in the Descripttion field.

Assign the functinality for the role by checking the permissions checkbox for the available functionality.

Click on Save Changes to Add the profile.

Edit profile

Select a role from the Select User Role drop down.

You will see the Role Name, Role Type and Decription as uneditable.

Assign or remove the functionality to the role by checking or unchecking the checkbox .

Click on Save Changes to Save the changes made to the profile or click Cancel Changes to revert back to the last saved information.

Delete Profile

Select a role from the Select User Role drop down which you want to delete.

Click on Delete Profile to delete the profile .

Note: The Portal Master Admin Profile cannot be edited or deleted.

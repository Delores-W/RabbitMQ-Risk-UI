Change Security Question

This screen lets you update the security question that you set to retrive the password, incase you forget the same.

Security Question: Choose any question from the Security Question drop down. If you forget the password at a later stage, you need to choose the same question in order to retrieve the password. Choose an easy question and do remember the question you have chosen in this field.

Security Answer: Enter the answer to the choosen question in this field. The answer needs to be a minimum of four characters. If you forget your password at a later stage, you will be required to answer the selected question in order to retrieve the password. Hence, do remember the answer you have typed in this field.

Password: Type your password in this field.

Click on Save Changes to update the Security Question or Cancel Changes to revert back to the original settings.

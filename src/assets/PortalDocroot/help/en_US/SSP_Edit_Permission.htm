Edit User Roles

In this screen, you can set the portal accessibility and company visibility of the sub account .

Portal Accessibility

You can set roles for each of the external applications for a particular sub account. An example: For the application “Portal” you can select the Available Roles and click on the to assign the roles to the sub account. Select the Roles which you want to remove from Assigned Role and click on to remove the asigned role for the sub user.

Company Visibility

Click on the Checkbox in the Access? column corresponding to the company to allow the user access to that company.

Click on Save Changes to save the changes made to Subaccount Permissions or Click on Cancel Changes to revert back to last saved information.

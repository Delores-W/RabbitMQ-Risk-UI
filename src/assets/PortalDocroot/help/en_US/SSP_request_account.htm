Help - myIATA

Request an Account

This screen allows you to request for an account in the myIATA portal.

Company Name: Type in your company name in this field.

IATA Numeric Code: Type the IATA Code in this field.

Address line 1: Type the contact address that will be used by IATA for correspondence.

Country: Select your Country from the drop down.

State / Province : Select your state / province from the drop down.

Contact Name: Type in the contact person's name in this field.

E-mail: Type in a valid email address, your temporary username and password will be mailed to this email id.

Type of Customer: Select the Type of Customer from the drop down.

Click on Submit. Your temporary username and password will be emailed you after the application has been successfully processed.

If you want to clear all the fields to start filling up the form again, click on Reset.

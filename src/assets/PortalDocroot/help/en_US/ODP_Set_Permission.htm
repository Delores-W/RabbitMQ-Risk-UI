Set Permission

You are in the Step 2: Set Sub User Roles and Company Visibility screen. On this screen, you can assign roles and set access rights of the new sub users to child companies. The roles may differ from one application to another. You can select roles for each application.

Available Roles shows the list of roles available for that user on a particular application. Select the roles from this field and click on the double forward arrow button.

Assigned Roles shows the list of roles that are assigned to that user. If you want to remove a role assigned to the user, select the role in the Assigned Roles field and click on the double backward arrow button.

The screen also gives you the list of child companies present under a parent company. The last field Access? Lets you assign the access . Click and a place a check mark in this field to allow the access. If there is no checkmark in this field, this indicates that the new sub user will not have access to that child company.

Click on Add to add a new sub account .

Add New T2005 Application

This screen lets you add a new T2005 application to the portal.

Application Name: Type the name of the application here.

Show Link As: Type the name of the application as you would like the customer to see it . An example: If you want an application by the name of CIS to appear as Finance in the portal, type in Finance in this field.

Link Position: Select the position at which the link appears in the top navigation. An example: if you choose the Link Position as 2, the new link will appear in the second position on the portal. If you do not choose any number in this field, by default the new application will be positioned last in the navigation bar.

Application URL: Provide the URL of the deployed application.

Portal Page URL: Provide the URL of the Portal page which is served to the customer before he can access the application.

Click on Add Application.

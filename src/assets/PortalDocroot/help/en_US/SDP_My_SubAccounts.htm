My Sub Accounts

This screen gives you the details of all the sub user accounts linked to the company which you have selected. The details include Username, Name, Address, Phone, Email etc.

The Second column Name has the names of all the sub users. Click on the name to look at the basic information of that sub user.

The last column on the screen, SUA Admin indicates whether a particular user is a Sub User Account administrator.
Indicates that the particular user is a SUA administrator.

You can add a new sub user account by clicking on the Add button.

To delete the sub account, select one or more sub accounts from the list and click on Delete button.

Forgot Password

This screen helps you to retrieve your password.

Username: Type in your Username in this field.

Security Question: Select the security question that you have set in Change Security Question page in Portal.

Security Answer: Type in the answer you have given in the Change Security Question page in Portal.

Click on Submit . Your password will be emailed to you.

If you want to clear all the fields, click on Reset .

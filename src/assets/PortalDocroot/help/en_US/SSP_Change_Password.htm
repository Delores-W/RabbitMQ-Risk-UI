Change Password

You can change your password in this screen.

Old Password: Type your existing password in this field.

New Password: Type in the password of your choice in New Password field. Password can be anywhere between 8 characters and 10 characters. The first seven characters should have atleast 1 capital letter. There should be atleast 1 number after the first 7 characters. E.g. 'Password1'

Retype new password: Confirm the new password by retyping the new password in this field.

Click on Change password to set the new password.

Click on Cancel Changes to empty all fields.

All the fields are mandatory; hence you can change the password only if you know the current password. It is recommended that you change the password at least once in three months.

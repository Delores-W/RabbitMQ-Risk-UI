IATA Operations Desk Portal Help

Select Parent

Process of adding a new external account has four steps: Step1: Select Parent , Step 2: Add Company Information , Step 3: Add Contact Information , Step 4: Set User Roles and Add Account.

You are in the Step1 : Select Parent Company screen.

If the company to which the external user belongs is a parent company, then select This Company is a Parent and click on Continue, which takes you to the next step to add the user's company information .

If the company to which the external user belongs is a child company, then choose Select Parent for this Company and click on Continue.

When you click on Continue , the screen gives an option to search for the parent company either by IATA Code , Agency Name , Country or City. Key in any of this information and click on Search Company.

From the search results , you can choose the parent company and click on Select. This takes you to the next step to add the user's company information.

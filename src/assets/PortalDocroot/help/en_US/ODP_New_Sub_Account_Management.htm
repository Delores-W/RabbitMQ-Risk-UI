Add New Sub Account Management

Adding sub accounts has 2 steps; Step 1: Sub User Contact Information and Step 2: Set Sub User Roles and Company Visibility .You are in Step 1: Sub User Contact Information screen. Providing the basic information of the new sub user on this screen.

Phone: Enter the new sub user's phone number in this field. Enter the country code in the first box followed by the area code and the telephone number in the next 2 boxes.

Fax: Enter the new sub user's fax number in this field. When you enter the number, follow the same format as you did to enter the telephone number.

Address line: Enter the new sub user's address in this field. This is the address which IATA uses for any communication with the user.

Username: Enter the new sub user's username in this field. The username should be same as his/her email address.

Email: Enter the new sub user's email address. It should be same as his/her username.

Click on Next. This takes you to Step 2: Set Sub User Roles screen where you can set roles for the new sub user.

Edit User Role – External User

You can reassign roles to an external user on this screen.

Available Roles shows the list of roles available for that user on a particular application. Select the roles which you want to assign from this field. Click on the double forward arrow button to assign those roles.

You can view the roles that you have assigned in the Assigned Roles field. If you want to remove a role assigned to the user, select the role in the Assigned Roles field and click on the double backward arrow button.

Click on Save Changes to Save the changes or Click on the Prev button to take you to the previous step without saving any changes.

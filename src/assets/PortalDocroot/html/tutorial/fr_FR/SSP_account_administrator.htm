Administrateur myIATA pour une société

Félicitations, vous avez été nommé administrateur de myIATA pour votre société. Veuillez trouvez ci-après la liste de vos responsabilités et tâches :

Responsabilités

  • Maintenir les coordonnées de votre société à jour.
  • Maintenir les informations de votre compte à jour.
  • Maintenir les sous-comptes nécessaires pour votre société à jour.
  • Accéder aux informations commerciales

Tâches initiales

1. Connectez-vous au portail en utilisant votre identifiant temporaire. Vous serez automatiquement invité à modifier votre identifiant temporaire.

  • Nom de compte (votre adresse e-mail),
  • Mot de passe,
  • Question de sécurité
  • Acceptez les conditions générales.

2. Indiquez vos coordonnées

3. Actualisez les coordonnées de votre société. Dans ces écrans, seuls certains champs peuvent être actualisés. Contactez le Service Clientèle si vous n'avez pas accès à des informations qui doivent être actualisées.

4. Créez des sous-comptes pour les autres employés de votre société. Une notification sera envoyée par e-mail aux nouveaux utilisateurs.

  • Accédez aux sous-comptes
  • Créez et gérez les sous-comptes de votre société.
  • Assignez des droits

Tâches en cours

  • Lorsqu'une personne quitte votre société, assurez-vous que son compte est désactivé.
  • Maintenez votre compte à jour (informations du compte, mot de passe, question de sécurité…
  • Maintenez les coordonnées de votre société à jour.

Accédez aux informations commerciales

Avec votre compte MyIATA, vous pouvez accéder aux informations commerciales.

  • Sélectionnez le domaine commercial dans le menu du haut.
  • Lisez la page de présentation pour en savoir plus sur les informations commerciales
  • Sélectionnez les liens appropriés

L'accès aux informations commerciales vous est accordé par l'administrateur du portail IATA.

Besoin d'aide ?

  • Dans chaque page de navigation, vous pouvez cliquez sur l'Aide se trouvant dans le menu situé en haut à droite. L'Aide vous fournira toutes les informations nécessaires.
  • Rencontrez-vous des problèmes pour accéder au site ? Allez dans la section Service Clients et envoyez une question.

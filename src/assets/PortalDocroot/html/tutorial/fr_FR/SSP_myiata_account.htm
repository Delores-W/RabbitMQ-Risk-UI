Compte myIATA

Félicitations, vous possédez désormais un compte dans myIATA. Veuillez trouver ci-après la liste de vos responsabilités et tâches :

Responsabilités

  • Maintenir les informations de votre compte à jour.

Tâches initiales

1. Connectez-vous au portail en utilisant votre identifiant temporaire. Vous serez automatiquement invité à modifier votre identifiant temporaire.

  • Nom de compte (votre adresse e-mail),
  • Mot de passe,
  • Question de sécurité
  • Acceptez les conditions générales.

2.Indiquez vos coordonnées

Tâches en cours

  • Maintenez votre compte à jour

Accédez aux informations commerciales

Avec votre compte MyIATA, vous pouvez accéder aux informations commerciales.

  • Sélectionnez le domaine commercial dans le menu du haut.
  • Lisez la page de présentation pour en savoir plus sur les informations commerciales
  • Sélectionnez les liens appropriés

L'accès aux informations commerciales vous est accordé par l'administrateur de votre société. Dans la page d'accueil du portail, le nom de votre administrateur est disponible dans le menu de gauche.

Besoin d'aide ?

  • Dans chaque page de navigation, vous pouvez cliquez sur l'Aide se trouvant dans le menu situé en haut à droite. L'Aide vous fournira toutes les informations nécessaires.
  • Ne voyez-vous aucune information commerciale ? Veuillez contacter l'administrateur de votre compte en cliquant sur son nom à partir du menu de gauche.
  • Rencontrez-vous des problèmes pour accéder au site ? Allez dans la section Service Clients et envoyez une question.

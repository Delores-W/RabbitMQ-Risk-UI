Registration Progress Report

The screen gives registration report in HTML format which in turn can be exported to Excel.

Export To Excel: When the report generated is in HTML format, click on Export to Excel which downloads the report in Excel format.

You may also print the report by clicking on Print.

Back: When you click on Back, it takes you back to the Registration Progress screen

SSP Account Consolidation Help

New Temporary Account

The screen gives your temporary account details.

Both Temporary user Name and Temporary Password fields are pre-filled and you cannot modify the details.

Click on Save Changes to save the changes or Back to cancel the changes made on the screen. When you click on Back Button, you go back to the Temp Account Management screen where in you can search for an agency.

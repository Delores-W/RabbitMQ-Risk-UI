SSP Account Consolidation Help

Create Temporary Accounts

This screen shows the number entities existing per country and allows you to create temporary accounts for each of the entities types. The entity types are Agency, Airline and Person & Persons.

Please select the country and the entity type for which you wish to create temporary accounts and click on Start to schedule the batch process. After the job is successfully registered, you cannot schedule another one till the job currently submitted / running is completed.

Click on the Country dropdown to select the country for which you want to create the temporary accounts.

Click on the Entity Type dropdown to select the entity type for which you want to create the temporary accounts for the selected country.

Select the Operation as Create if you want to create temporary accounts for a new country and its entity or the first time. Or select the Operation as Create Failed if you want to create temporary accounts for a country and its entity for which some temporary account creation has failed in a previous batch process.

Click on Enter.

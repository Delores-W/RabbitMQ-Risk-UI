Export Temporary Accounts To Excel Sheet

This screen shows the number of entities per country for which temporary accounts have been created and allows you to export temporary accounts for each of the entities types. The entity types are Agency, Airline and Person.

Please select the country and the entity type for which you wish to export temporary accounts and click on Export to Excel. A windows prompt will appear requesting you to Open or Save the document. Please save this Excel document in a suitable location.

Click on the Country dropdown to select the country for which you want to export the temporary accounts.

Click on the Entity Type dropdown to select the entity type for which you want to export the temporary accounts for the selected country.

Click on Export to Excel.

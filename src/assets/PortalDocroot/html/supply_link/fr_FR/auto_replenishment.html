Réapprovisionnement automatique

Dans le cadre du projet Supplylink :

  • Les algorithmes utilisés par le réapprovisionnement automatique ont été révisés dans ISIS et dans Maestro afin de s'assurer qu'ils prennent constamment en compte les cycles saisonniers et les tendances.
  • Les fonctionnalités des commandes Automatisées / Manuelles / Urgentes ont été améliorées.

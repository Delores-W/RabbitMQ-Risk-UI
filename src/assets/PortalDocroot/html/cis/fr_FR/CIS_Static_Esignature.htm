À propos des factures électroniques

Vais-je recevoir des copies papier de certaines factures?

Actuellement, seules les factures générées par notre fournisseur DPC pour la distribution via BSPLink peuvent être signées électroniquement.  Vous recevrez donc des copies papier pour les autres factures générées par BSP.  Même si des copies électroniques de la facture peuvent déjà être disponibles, seules les copies papier sont valables en tant que factures avec TVA. Celles-ci doivent donc être retenues.  À l'avenir, toutes les factures IATA seront signées électroniquement, et les copies papier ne seront alors plus nécessaires.

Dois-je conserver une copie papier de mes factures électroniques?

Non, il n'est pas nécessaire de conserver une copie papier des factures signées électroniquement, mais vous devez sauvegarder la version électronique.  Si vous êtes soumis à un contrôle fiscal ultérieurement, les autorités fiscales voudront simplement voir le document signé électroniquement.

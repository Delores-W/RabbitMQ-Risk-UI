Retrive Password

Purpose:

  • Allows user to retrieve forgotten password.

Actions:

  • Click on Forgot you Password? to invoke assistance to retrieve password.
  • Window with a Password Retrieval Form is generated.
    1. Enter Username
    2. Select Security question form the drop down menu
    3. Security Answer
    4. Enter the code as you see in the box at the bottom
  • Click on Submit to initiate validation of details. On confirmation, auto generated password will be sent to the user.
  • Click on Back to go back to the login screen.
  • Click on Help for assistance regarding retrieval of passwords.

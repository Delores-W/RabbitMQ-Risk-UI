Temporary Accounts

Purpose:

Temporary Account Management assists in

  • Tracking the status of accounts for various entities (registered, temporary or no account)
  • Create New Temporary Accounts
  • Reset password for users

Â 

Actions:

Search for an Account

  • Enter any one or more search criteria. Click on Search to get the search results.
  • Click on Help for assistance on the Account consolidation as a pop-up window.
  • Click on Back to Portal to return to SDP default page (Company/User Search).

Special Cases in search :

Search CategoryDisabled search fields/actions
Agency Nil
Airlines Agency Status
Others IATA#, Agency Status

View Status

  • Click on View Status for a company to go to view status page that displays current situation for that account. The status of an account can be:
    • Registered
    • Temporary
    • No Account
  • Click on Back to return to Search Results page.

View Password

  • Select the radio button for the account and Click on View Password to view the Temporary User Name and Password.
  • Click on Reset Password to impose a password change for the temporary account user.
  • The Temporary Username and Password is generated.
  • Click on Back to return to Search Results page.
  • Password cannot be reset for:
    • Account Status as Registered (registered Users)
    • Account with multiple temporary users
    • No User exists

New Temporary Account

  • Select the radio button for the account and click on New Temporary Account to create Temporary User for that account.
  • User name and Password is generated. Click on Save Changes to complete creating an account.
  • Click on Back to return to Search Results page.

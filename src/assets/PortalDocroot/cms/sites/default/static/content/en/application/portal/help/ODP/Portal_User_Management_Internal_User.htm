Portal User Management - Internal User

Purpose:

This screen permits an ODP user to manage the internal user accounts which includes

  • Viewing User Contact Information
  • Assigning User Roles for portal accessibility
  • Setting Geography for office access

Actions:

Search

  • Enter any 1 or more of the search criteria:
    • First Name
    • Last Name
    • User Name
    • Click on Search to get the search results. The system will generate the results in below format.
    • Click on Employee name to view contact details

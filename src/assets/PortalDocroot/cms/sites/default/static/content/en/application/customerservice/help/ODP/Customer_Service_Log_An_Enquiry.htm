Help - myIATA Operational Desk Portal

Log an Enquiry

Purpose:

  • This screen enables a user to log a query with Customer Service of IATA to seek appropriate solutions.
  • This can be tracked through the View enquiry module under Customer Services.

Actions:

  • User needs to fill an enquiry form.
  • General User Information is pre populated
  • User needs to enter information like case area, subject, description etc.
  • Click on Submit Query to send question to IAT customer service..

Temporary Accounts - Registration Progress Report

Purpose:

User can generate reports on the registration progress of the temporary accounts created for different sate ranges.

The reports can be generated in HTML or Excel format.

Actions:

  • Select the filter criteria to generate the report:
    • Dates between which the accounts were created
    • Country for which the accounts were created
    • Status of the account Registered/ Un-registered
    • Specify the Format for viewing the report
  • Click on Reset to clear all the selections and start afresh.
  • Click on Generate Report to view the progress report.
  • Click on Back to return to ODP User Home for Temporary Accounts page.

Portal Monitoring - Platform Logs

Purpose:

This is a view only screen that tracks all the Account Creation related activities on portal. It captures vital information like IP address, login date, etc. This assists in following up failed attempts.

Actions:

  • Search for a record by entering the relevant information in any of the filter criteria.
  • Click on Search .
  • Click on Reset to reorganize the search criteria.
  • Click on Export to Excel to copy the above table into an excel format for further modifications (if any).
  • To view next page, click on next button on the bottom left corner of the display. Click on the page number to view a specific page.

Table Contents:

Header NameDescription
IATA Code Unique Id number given to each account
Username Username or the email ID for the individual users
Company Name Gives the name of the company to which the account belongs to
Country Gives the Country name to which the company belongs
City Gives the city name to which the company belongs
Login Date Gives the last login date and time of individual user
Status Gives the current status of the New account created in terms of their success or failure
User Created Date Gives the date on which the user was created
IP Address Gives the IP address specific to the individual

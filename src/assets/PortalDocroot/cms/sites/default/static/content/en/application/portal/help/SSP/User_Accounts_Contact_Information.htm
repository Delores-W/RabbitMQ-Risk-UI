User Accounts - Contact Information

Actions:

  • User can enter or edit the Basic user Information.
  • The last 3 fields are read only.
  • Click on Save Changes to save changes.
  • Click on Cancel Changes to discard all actions.
  • User may also click on the hyperlinks on the top navigation to move to next step.

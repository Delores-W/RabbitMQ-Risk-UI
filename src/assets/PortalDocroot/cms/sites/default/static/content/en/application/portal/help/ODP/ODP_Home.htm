Opertional Desk Portal Homepage

Purpose:

  • This page is shown after the user has successfully logged into the portal.
  • This page shows downtime information if available. Details of next scheduled downtime (displayed only 24hrs before the schedule time) are displayed with a warning indicating the Application name, date and time.
  • Multiple downtimes can also be displayed separately in the same area.

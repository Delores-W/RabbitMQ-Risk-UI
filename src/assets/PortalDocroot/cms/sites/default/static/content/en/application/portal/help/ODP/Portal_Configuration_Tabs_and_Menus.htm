Portal Configuration - Tabs and Menus

Purpose:

The screen allows user to add or regroup the applications to the menu of existing tabs in the portal (at the top menu).

User can also add New Tab names and assign applications to it.

Actions:

  • Select the tab type from the drop down, it will give options of existing tabs or an option to create a new Tab .
  • To assign an available application to that tab, select the application and click on the copy button
  • To remove an assigned application, select the application and click the remove button
  • Click on Save Changes to update the lists.
  • Click on Cancel Changes to discard all actions.
  • Tab Name would appear only if the user has selected Add New Tab.

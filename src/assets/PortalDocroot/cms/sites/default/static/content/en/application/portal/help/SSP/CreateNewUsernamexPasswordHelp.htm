Create New Username and Password

You can create new SSP user account and password on this screen.

New username: Please enter a valid email id as Username. You will receive the notification at this email id on completion of the registration process.

New password: Password can be anywhere between 8 letters and 10 letters with 1 capital letter in the first 7 letters and 1 number after that.

Confirm the password by retyping it in the Retype Password field.

Security Question: Choose any question from the Security Question drop down. If you forget the password at a later stage, you need to choose the same question from this field in order to retrieve the password. Choose an easy question and do remember the question you have chosen in this field.

Security Answer: Enter the answer to the chosen question in this field. Enter the answer to the chosen question in this field. The answer needs to be a minimum of four characters. If you forget your password at a later stage, you will be required to answer the selected question in order to retrieve the password. Hence, do remember the answer you have typed in this field.

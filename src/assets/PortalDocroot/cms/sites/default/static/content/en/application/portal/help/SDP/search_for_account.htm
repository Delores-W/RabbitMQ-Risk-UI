Search for an Account or User

Purpose:

This is the default login screen, allows user to search for an account.

Actions:

  • Select the search group, it can be any one of the below:
    • Company
      1. Agency
      2. Airlines
      3. Others
    • Person
  • Enter any one or more search criteria. Click on Search to get the search results.
  • Company Search: When the Search category is a company.
    1. Number of records to be displayed per page can be set from the drop down on top right corner of the search results.
    2. To view the company information, click on Company Name.
  • Person Search: When the Search category is a person.
    1. First Name, Last Name and Username Search Criteria will appear.
    2. Enter any of the search criteria and click on Search Account .
    3. Clicking on the Person Name will open User Contact Information.

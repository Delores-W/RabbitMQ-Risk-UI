Portal User Management - External User

Purpose:

This function of this screen is to allow an ODP user to manage the external user accounts.

External user (Company or person) account details can be viewed and edited, that includes

  • Step 1: Company Information
  • Step 2: Take Trust
  • Step 3: Give Trust
  • Step 4: User Accounts
  • Add New Account

ODP can build trust or give trust to different types of users.

This screen also manages the User accounts for companies and adds New Accounts for different companies. This screen can administer

  • Step 1: User Contact Information
  • Step 2: User Roles
  • Step 3: Company Visibility
  • Add New User

Actions:

Search

  • Select the search group, it can be any one of the below:
    • Agency
    • Airlines Company
    • Others
    • Person
  • Enter any one or more search criteria. Click on Search to get the search results.
  • Company Search: When the Search category is a company.
    • Example: For Agency search the system generates search results in below format
    • Number of records to be displayed per page can be set from the drop down on top right corner of the search results.
    • Select the relevant radio button and click on Disable Account to deactivate that account. The status will be changed from active to Disabled.
    • To view the company information, click on Company Name.
  • Person Search: When the Search category is a person.
    • First Name, Last Name and Username Search Criteria will appear.
    • Enter any of the search criteria and click on Search Account .
    • Clicking on the Person Name will open User Contact Information

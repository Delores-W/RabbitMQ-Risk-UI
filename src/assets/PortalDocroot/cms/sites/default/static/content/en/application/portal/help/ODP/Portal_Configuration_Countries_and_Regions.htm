Portal Configuration - Countries and Regions

Purpose:

The user can configure live countries for a particular region for myIATA.

List of Available Countries for each region is and the Live Countries where the application is live are displayed.

User can add or remove countries from the Live Countries list.

Actions:

  • Select a region to view the list of available countries.
  • To add an available country to Live Countries, select the country and click on the copy button
  • To remove a live country from the list, select the country and click the remove button
  • Click on Save Changes to update the lists.
  • Click on Cancel Changes to discard all actions.

Applications Events - TreasuryLink Events

Purpose:

This screen t racks the success or failure of all the actions/events on TreasuryLinks application.

The failed events can be rescheduled.

Actions:

  • Search for an event by entering the date of failure of action.
  • Click on Go button to view the search results.
  • You can click on the date of the event to see the details .
  • Status denotes the failure or success of the action.
  • Click on Reschedule button to immediately reorganize the event.
  • Reschedule button will not be available if the status of the event is success. It would be for failed events only.

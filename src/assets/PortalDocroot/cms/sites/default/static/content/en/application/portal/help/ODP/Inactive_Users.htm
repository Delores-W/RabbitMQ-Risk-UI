Inactive Users

Purpose:

Once a user had not been using his account for 90days, the user becomes Inactive.

This is a view only screen that assists in tracking inactive users.

Actions:

  • Search for a user by entering the relevant information in any of the filter criteria.
  • Created Date: date on which the account was created. (This has to be atleast 90days prior to the current date).
  • Last Login Date: date on which the user had last logged in to the account.
  • Click on Search to view the list of Inactive Users for the selected criteria.
  • Click on Delete User to remove an inactive user(s) by marking the applicable check box(s).

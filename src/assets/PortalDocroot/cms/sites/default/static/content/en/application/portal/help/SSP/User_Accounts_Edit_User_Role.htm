User Accounts - Edit User Role

Actions:

  • Click on or Step 2: Edit User Roles in the middle top navigation to go to edit screen.
  • Available roles specific to an application and assigned roles to that user will be displayed.
  • To assign an available role to that user for any application, select the role from left column and click on the copy button
  • To remove an assigned role from any application, select the role from the right column and click the remove button
  • Click on Save Changes to save changes.
  • Click on Cancel Changes to discard all actions.

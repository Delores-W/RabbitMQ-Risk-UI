Help - myIATA Operational Desk Portal

Login

Purpose:

This page is for the ODP user to login to the myIATA portal. The user has to enter the username/password to login into the system. Besides this the page also shows scheduled downtimes. Details of the next scheduled downtime are displayed with a warning stating the

  • Application name
  • Date & time of downtime (always in Geneva Mean Time)

Scheduled Downtime information will appear only 24hrs before. If there is no schedule downtime then that space is empty.

Actions:

  • Enter the username in the Username field.
  • Enter the password in the Password field.
  • Click on Login that takes you to the home page.
  • If the username/password entered is valid then user is directed to the ODP home page.
  • If the username is invalid then error message is displayed.

Company Account - Company Information

 

Purpose:

  • This screen displays User Company information and invoicing details.
  • Most information is read only, few fields can be edited
    • Email
    • Sales Tax Area Code
    • Invoice Delivery Method (user can make it e-invoice by selecting the check box)
    • Invoice Currency
    • Invoicing Language

Tips:

  • Please contact us if changes are required in the company non editable fields.
  • Click on Save Changes to update the user records.
  • Click on Cancel Changes to discard all actions. /ul>

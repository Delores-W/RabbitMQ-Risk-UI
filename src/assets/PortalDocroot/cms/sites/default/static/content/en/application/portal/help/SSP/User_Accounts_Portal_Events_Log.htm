Portal Event Log

Purpose:

This is a view only screen that tracks Portal external events like creation of account, reset password, sales white mail related actions, etc.
SDP can search for events and track their success or failure. They may also export the list of events to excel sheet for further alterations.

Actions:

  • Select the search group, it can be any one of the below:
    • Company
      1. Agency
      2. Airlines
      3. Others
    • Person

Company Search: When the Search category is a company.

  • Search for an event by entering the relevant information in any of the filter criteria.
  • Click on Search .
  • The search results will be in the below format.

Person Search : When the Search category is a person.

  • First Name, Last Name and Username Search Criteria will appear.
  • Enter any of the search criteria and click on Search.
  • The search results will be in the below format.
  • Click on Export to Excel to copy the above table into an excel format for further modifications (if any).
  • To view next page, click on next button on the bottom left corner of the display. Click on the page number to view a specific page.

Table Contents

Header NameDescription

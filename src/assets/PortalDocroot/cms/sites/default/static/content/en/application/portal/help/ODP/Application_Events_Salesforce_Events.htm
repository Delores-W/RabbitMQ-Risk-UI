Applications Events -Salesforce Events

Purpose:

For every successful account create d, a SalesForce account is also created. This screens tracks the failed attempts in creating the SalesForce account citing the reason for the same. ODP user can initiate the R e-creation of failed account creation.

Actions:

  • Search for an event by entering the relevant information in any of the filter criteria.
  • Click on Search Account to search for the failed SalesForce Events and their details.
    • First Name of a user
    • Last Name of a user
    • Date s within which the creation of account was attempted by the system
    • Username of the individual
  • The reason for the failure in the account creation is displayed.
  • Once the flaw is corrected, click on Create Account button to schedule the redo the create account process from the below tab.
  • The scheduling of this process can be performed in 3 ways by selecting the appropriate radio button:
    • Now: to immediately start creating the account again.
    • Default: to schedule the account creation for the default night time.
    • Custom: to schedule the account creation at a later date and time.
  • Click on Schedule button to record the scheduling.

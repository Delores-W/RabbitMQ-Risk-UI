Help - myIATA Support Desk Portal

SDP Navigation Tools
  • Top Navigation allows user to access various related applications like accrediation, invoices, BSPlink, Ticket Supplies, Customer service, etc.
  • Left Navigation allows ODP user to supervise and monitor user and role based functional activities on the portal.
  • The top right Toolbar is available on all screens. Each item is explained below.

  • User can print the displayed page.
  • Click on PRINT to open up print box and make relevant selections. Click on Print to print the page.

  • Click on the SEARCH IATA button. This opens up a search box; enter the search text.

  • Click on Search to view search results in a new window.

  • HELP icon assists users to get help on the details of the page.
  • Click on HELP , to get detailed description on the contents of the page.

  • User can contact IATA for any information.
  • Clicking on CONTACT US, brings up a pop-up asking user to fill details on 3 aspects:
    • Case Area; related to the query
    • Subject
    • Description
  • Click on Submit to send the query to IATA.
  • Click on Reset to clear and reorganize the query to be sent to IATA.


  • Clicking on LOGOUT logs user out of the myIATA application and goes back to the login screen. A window asking if "Are you sure, you want to log off" will come up. Click on OK to log off and cancel to resume.

  • This tab is available only when the User is in Proxy User Mode .
  • Clicking on DISCONNECT logs user out of the User/company account that was being accessed and takes user back to the search screen.

Request an Account - PRIN Validation

Actions:

  • Validate the information (edit/enter details if required), enter the code as you see in the box at the bottom and click on Submit .
  • On submitting, Account Creation Form is generated.
  • Fields marked (*) are mandatory.
  • Click on Submit to update the information entered and validate.
  • Click on Reset to clear all the information entered.
  • Bring the cursor close to question mark for tool tips.

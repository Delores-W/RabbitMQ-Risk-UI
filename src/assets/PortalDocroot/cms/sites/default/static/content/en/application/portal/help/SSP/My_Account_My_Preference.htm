Help - myIATA Self Service Portal

My Account - My Preference

Purpose:

This screen allows SSP user to set their preferences to receive IATA generated content through emails in the desired language.

Actions:

  • Choose a Language from drop down menu.
  • Set your preferences by selecting the appropriate check boxes.
  • Click on Save Changes to update the lists.
  • Click on Cancel Changes to discard all actions.
  • A message Preferences have been set will appear on saving changes.

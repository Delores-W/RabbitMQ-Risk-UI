Portal Events

Purpose:

This is a v iew only screen that tracks the internal events like Mass Mailing, Treasury Links - Data transfer, Role Management, etc. This assists in following up failed events.

Actions:

  • Search for an event by entering the relevant information in any of the filter criteria.
  • Click on Search .
  • Click on Export to Excel to copy the above table into an excel format for further modifications (if any).
  • To view next page, click on next button on the bottom left corner of the display. Click on the page number to view a specific page.

Table Contents

Header NameDescription
Username Username or the email ID for the individual users
Event Gives the event that has occurred on the portal for the particular user
Initiated By Gives the name of the person who had initiated the event
Initiated From Gives the application type from which the action/event was commenced
Date/Time Gives the exact date and time of the occurrence of the Event.
Status Gives the current status of the event in terms of their success or failure

Account Management - Role Compatability

Purpose:

ODP user can set compatible roles across all available applications, depending on the type of user.

The settings on this screen form the basis for Role and Profile definitions.

Actions:

  • Select the User Type from the drop down menu.
  • Select the Registration Type from the drop down menu, that depends on the user type chosen
    • Not available for PRIN and Public User
    • Mandatory for Company User
  • Check/ Uncheck boxes to set the role compatibility.
  • Click on Save Changes to update the lists.
  • Click on Cancel Changes to discard all actions.

Temporary Accounts - Create Accounts

Purpose:

This screen shows the number entities existing per country and allows ODP User to create temporary accounts for each of the entities types by Registering a Job The entity types are Agency, Airline and Person & Persons

Once a job is successfully registered, another one cannot be schedules till the job currently submitted / running is completed .

User can view the status of job registered .

Actions:

Register Job

  • Create temporary accounts by selecting the Country name and Entity Type .
  • Select the operation for which you wish to register the job
  • Create: to schedule a job for creating the temporary account
  • Create Failed: to re-schedule the job for failed creation of temporary accounts.
  • Click on Register Job to schedule creation of user accounts.
  • Click on View Job Status to view the current position of the previous job registered.
  • Click on Back to return to ODP User Home for Temporary Accounts page.

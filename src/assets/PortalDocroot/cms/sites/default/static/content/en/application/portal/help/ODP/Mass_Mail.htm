Mass Mail - Select Recipient

Purpose:

This screen enables ODP user to send Mass mails to all users or specific set of users for

  • Announcements
  • Notifications

This screen facilitates ODP user to send the group mails to different types of recipients like Company, super Users or all users.

Action:

Step 1: Select Recipient

  • Select the search group, it can be any one of the below:
    • Agency
    • Airlines
    • Others
    • Person
  • Enter any one or more search criteria. Click on Confirm to get the user list for that account. The system will generate the results table in below format.
  • Refer to Mass Role Assignment Step 1: Select Assignee; the search procedure is same as selecting a Recipient.
  • Only the total number of records for the selected search criterion is displayed.
  • Select the applicable radio button from the Select Recipients Tab . Click on Send to go to next step and begin framing the mail for all chosen recipients.
  • Please Note: If the search group selected is a Person, the Select Assignee Tab would not appear.

To Edit the Recipients List

  • Click on Download Records to view the entire list to edit it.
  • Save the file in CSV format and make required changes.
  • Click on Upload CSV to upload the saved list of assignees. An Upload Tab will be displayed.
  • Click on Browse to select the file to upload.
  • Click on Upload to select the new list as the Assignees.
  • Click on Send to begin framing the mail for this list of users.

User Accounts

Purpose:

The User accounts can be managed from this screen assigning roles and defining the company visibility. SDP proxy user can search for users and for each user manage the

  • User Contact Information
  • User Roles
  • Company Visibility
  • Create/Add New User

At times users may forget their password or lock accounts due to success logins with incorrect password, SDP can

  • Reset Password
  • Unlock user accounts

Actions:

  • Search results are pre-listed; user may enter any one or more of the search criteria for further filtering and Click on Search .
  • The tick mark in the Admin column indicates that the user has admin rights.
  • If a User has locked their account by using incorrect password repetitively
    • Select the check box, click Unlock Account to release that account.
    • Select the check box; click Reset Password to induce the user to change the password in the next login. Auto generated password is mailed to the user.
  • To terminate an account, select the check box of the user and click Delete .
  • Click on User name to view/edit user Contact Information.

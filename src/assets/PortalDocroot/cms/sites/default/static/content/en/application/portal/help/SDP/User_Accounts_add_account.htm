User Accounts - Add New Account

Purpose:

This screen allows SDP User to create/add new user to the account.

Actions:

  • Before Entering User contact information, please select
    • Company
    • User Type (administrator/sub user)
    • Access Type
      1. For Full access , you need to enter only contact information, other steps are not required (not visible)
      2. For Restricted access , you need to specify roles and company visibility.
  • All the fields are editable and mandatory fields are marked.
  • To move to next screen, user must enter all mandatory fields and
    1. For Full access , the check box to accept terms & conditions and click on Add Account .
    2. For Restricted access , click on Add Account .

Restricted access

  • Edit User Role
    • Available roles specific to an application and assigned roles to that user will be displayed.
    • To assign an available role to that user for any application, select the role from left column and click on the copy button
    • To remove an assigned role from any application, select the role from the right column and click the remove button
    • Click on Next to go to next step.
    • Click on Back to return to previous step.

Company Visibility

  • Current company of the user is highlighted in a different color.
  • User may grant access to other companies in the network by selecting the relevant check box.
  • Select the check box to accept terms & conditions and click on Add Account to complete addition of new user.
  • Click on Back to return to previous step.

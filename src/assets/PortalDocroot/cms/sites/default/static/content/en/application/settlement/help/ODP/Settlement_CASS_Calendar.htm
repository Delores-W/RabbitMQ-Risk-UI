CASS Calendar

Purpose:

  • This screen allows user to view CASS Calendar for a specific country and particular year.

Actions:

  • Select a country name (mandatory field) from the drop down menu.
  • Select a year from the drop down menu to view the respective calendar.
  • To view next page, click on next button on the bottom left corner of the display. Click on the page number to view a specific page.

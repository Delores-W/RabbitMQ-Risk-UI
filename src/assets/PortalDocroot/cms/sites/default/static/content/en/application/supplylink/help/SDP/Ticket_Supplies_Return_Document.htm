Return Traffic Documents

Purpose:

  • This page has the form which is to be used to request a pick up or to return a certain stock of Standard Traffic Documents or ticket wallets.
  • This can be tracked through the View enquiry module under Customer Services.

Actions:

  • General Information will be fetched from the database, these may be edited.
  • Fill the form and Click on Submit Query to sent the form to IATA.

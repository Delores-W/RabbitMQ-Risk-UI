Help - myIATA Self Service Portal

Login

Purpose:

  • This page is for the SSP user to login to the myIATA portal. The user has to enter the username/password to login into the system.
  • Allows the SSP user to retrieve forgotten password.
  • For a new user, this screen is the pathway to request for an account with IATA.
  • This page also shows scheduled downtimes. Details of the next scheduled downtime are displayed with a warning stating the
    1. Application name
    2. Date & time of downtime (always in Geneva Mean Time)
  • Scheduled Downtime information will appear only 24hrs before. If there is no schedule downtime then that space is empty.
     


Actions:

Login to myIATA

  • Enter the username in the Username field.
  • Enter the password in the Password field.
  • Click on Login which takes you to the home page.
  • If the username/password entered is valid then user is directed to the Company Search.
  • If the username is invalid then error message is displayed.

Airline Contact Details per market

Purpose:

  • This screen enables users to find the contact details of the airline offices responsible for CASS and BSP activities in different markets .

Actions:

  • Select any of the search criteria to extract list of airlines.
  • Select the country form the drop down menu (mandatory field).
  • Click on Search to get the results.
  • The list will be displayed with above details.

Table Contents:

Airline Name Name of the Airline
IATA Designator 2 letter IATA description assigned to the airline

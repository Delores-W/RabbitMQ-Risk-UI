Temporary Accounts - Summary of Temporary Accounts

Purpose:

This screen displays the Summary of Temporary Accounts for White Mail. It shows the number of entities per country for which temporary accounts have been created and allows User to export temporary accounts to an excel sheet for each of the entities types. The entity types are Agency, Airline and Person .

The list of users may be edited for an entity.

Actions:

  • Select the Country name and Entity Type to view the summary.
  • Click on Export to Excel to reproduction the table in an excel format for print or edit purpose.
  • Click on Back to return to ODP User Home for Temporary Accounts page.
